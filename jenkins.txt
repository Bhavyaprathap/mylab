Before creating the pipeline:

Jenkins installed and running on localhost.

Git Plugin

1.2 Tool Configuration

Go to:
Manage Jenkins → Global Tool Configuration

Configure:

JDK (e.g., JDK-17)

Maven (e.g., Maven-3.9)

Git

⭐ 2. GitHub Maven Project

The GitHub repository must contain:

pom.xml

src/main/java/...

(Optional) Jenkinsfile

Example repo used in exam:

https://github.com/<username>/<MavenJavaProject>.git

⭐ 3. Create a Jenkins Pipeline Job
Steps:

Jenkins Dashboard → New Item

Name: JavaBuildPipeline

Select Pipeline

Click OK

⭐ 4. Write the Jenkinsfile

You either:

Paste it directly into Jenkins, or

Save it in your repo as Jenkinsfile

✔ Full Jenkinsfile Solution (Exam-Ready)
pipeline {
    agent any

    tools {
        maven 'Maven-3.9'   // Name configured in Jenkins
        jdk 'JDK-17'        // Name configured in Jenkins
    }

    stages {
        stage('Clone Repository') {
            steps {
                git 'https://github.com/<username>/<MavenJavaProject>.git'
            }
        }

        stage('Build') {
            steps {
                sh 'mvn clean install'    // Linux labs
                // bat 'mvn clean install' // Windows labs
            }
        }

        stage('Run Application') {
            steps {
                sh 'java -cp target/*.jar com.example.App'
            }
        }
    }

    post {
        success {
            echo "BUILD SUCCESS and Application Executed Successfully!"
        }
        failure {
            echo "BUILD FAILED!"
        }
    }
}


✔ This Jenkinsfile performs all required tasks:

Clone

Build using Maven

Run the Java program

Display success message

✅ 1. Prerequisites & Environment Setup

Before creating the pipeline:

1.1 Install & Configure Jenkins

Jenkins installed and running.

Required plugins:

Pipeline

Git

Deploy to Container Plugin

1.2 Configure JDK and Maven in Jenkins

Go to:
Manage Jenkins → Global Tool Configuration

Add:

JDK 17

Maven 3.x

1.3 Configure Apache Tomcat

Install Apache Tomcat (v8/9/10)

Start server:
http://localhost:8080

Create a Tomcat manager user in tomcat-users.xml:

<user username="admin" password="admin123" roles="manager-script,manager-gui"/>


Restart Tomcat.

1.4 Add Tomcat Credentials in Jenkins

Manage Jenkins → Credentials → Global → Add Credentials

Username: admin

Password: admin123

ID: tomcat-creds

⭐ 2. Create Jenkins Pipeline Job

Dashboard → New Item

Name: WebAppPipeline

Select Pipeline

Click OK

⭐ 3. Configure GitHub Repository

Under Pipeline → Definition: Pipeline script
Add your GitHub repo URL inside the Jenkinsfile.

⭐ 4. Jenkinsfile (Full Pipeline Script)

This is the script you paste inside Jenkins OR save as Jenkinsfile in GitHub.

pipeline {
    agent any

    tools {
        maven 'Maven-3.9'   // Name configured in Jenkins
        jdk 'JDK-17'
    }

    stages {

        stage('Clone Repository') {
            steps {
                git 'https://github.com/<username>/<web-project>.git'
            }
        }

        stage('Build WAR') {
            steps {
                sh 'mvn clean package'   // Linux
                // bat 'mvn clean package' // Windows labs
            }
        }

        stage('Deploy to Tomcat') {
            steps {
                deploy adapters: [
                    tomcat9(credentialsId: 'tomcat-creds', path: '', url: 'http://localhost:8080')
                ],
                war: 'target/*.war'
            }
        }
    }

    post {
        success {
            echo 'Deployment Completed Successfully!'
        }
        failure {
            echo 'Build or Deployment Failed!'
        }
    }
}


✔ Clones
✔ Builds WAR
✔ Deploys to Tomcat
✔ Shows status

Build Now
